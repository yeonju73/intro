# I AM

## 기획

본 기획안은 부트캠프 동료 및 강사님들을 타겟으로 하여, 전문성과 협업 능력을 강조하는 방향으로 설계되었습니다.

| 구분 | 상세 내용 |
| :--- | :--- |
| **1) 캐치프레이즈 후보** | 1. 금융의 견고함과 일상의 즐거움을 잇는 개발자<br>2. 안정적 백엔드 구조로 더 나은 내일을 설계합니다<br>3. 끊임없이 연구하고 함께 성장하는 백엔드 개발자 박연주입니다 |
| **2) 매력 포인트** | 1. **금융 도메인 특화:** 안전한 금융 서비스를 구현하기 위해 시스템 설계의 안정성을 깊이 고민하는 태도<br>2. **실행력 있는 연구 정신:** 막히는 기술적 이슈를 끝까지 파고들어 해결책을 도출하는 끈기<br>3. **협업 적합성:** 팀 프로젝트 시 ISTP 특유의 논리적 사고와 동료를 돕는 원만한 협업 능력 |
| **3) 타겟 방문자** | 부트캠프 동료 수강생 및 강사님 (기술적 열정과 협업 가능성을 보여줄 동료로서의 인상) |
| **4) 스토리텔링 흐름** | [도입] 직관적인 자기소개 → [본문] 전문성을 보여주는 About Me → [역량] 강점(구조, 연구, 협업) 시각화 → [여백] 친근함을 더하는 취향 → [결론] GitHub 링크 |
| **5) 무드/톤** | 차분한 (Calm), 신뢰감 있는 (Reliable), 세련된 (Sophisticated) |
| **6) 메인 컬러 + 폰트** | **메인 컬러:** `#2D3A2D` (Deep Forest Green) & `#5D7A55` (Sage Green)<br>**폰트:** Pretendard (가독성 높은 깔끔한 Sans-serif) |


## 구현

### 📋 프로젝트 소개

본 프로젝트는 부트캠프 동료 및 강사님들께 저의 기술적 역량과 협업 가치관을 효과적으로 전달하기 위해 기획된 개인 포트폴리오 웹사이트입니다. 금융권 IT 도메인을 목표로 하는 개발자로서, 신뢰감 있고 깔끔한 인터페이스를 통해 저의 강점을 직관적으로 보여주는 데 중점을 두었습니다.

### 🛠 기술 스택

이 포트폴리오는 외부 라이브러리 의존성을 최소화하고 웹 표준 기술만을 활용하여 제작되었습니다.

- HTML5: 시맨틱 마크업을 통한 구조 설계

- CSS3: Tailwind CSS를 활용한 스타일링 및 커스텀 애니메이션 구현

- JavaScript (Vanilla): IntersectionObserver를 이용한 순차적 페이드인(Fade-in) 인터랙션 구현

### 💡 주요 내용

- About Me: 백엔드 개발자로서의 전문성과 성장 지향적인 태도를 소개합니다.

- My Strengths: 안정적인 구조 설계, 연구 정신, 협업 마인드 등 개발자로서의 핵심 역량을 강조합니다.

- Life & Taste: 개발 업무 외의 친근하고 일상적인 모습을 보여주어 '함께 일하고 싶은 동료'의 이미지를 전달합니다.

- GitHub: 상세 프로젝트 이력 및 코드를 확인할 수 있는 외부 링크를 제공합니다.

### 🎨 디자인 시스템

- 메인 컬러: Deep Forest Green (#2D3A2D) & Sage Green (#5D7A55)

- 폰트: Pretendard (가독성과 현대적인 느낌을 주는 Sans-serif 서체)

- 무드: 차분함, 신뢰감, 세련미
